== Description ==

Put a widget with a image gallery slide show.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After yo activate the plug in, you will see an new option called "WP Image Show".

== Selecting the images ==
5. Go to the "WP Image Show" option in the Settings menu.
6. Select the images you want to include in your gallery and click  "save gallery information".

== Activating the widget ==
   
8. Once you have selected the images, go to the "Widgets" option under Design/Appearance menu.
    
9. Search for the "WordPress Image Show" and click "add". The widget will be added on the
   selected widget list.
   
10. Click edit for set the width, height and time elapsed between images. 
11. Done !
